# WeatherDiaryProject
WeatherDiaryProject; hava durumu verilerinin kaydedilmesi, görüntülenmesi ve analiz edilmesi amacıyla geliştirilmiş çok katmanlı bir web uygulamasıdır. Proje; ASP.NET Core MVC, Web API, gRPC ve Node.js servislerini bir arada kullanarak modüler ve ölçeklenebilir bir mimari sunar.
Bu proje, katmanlı mimari ve servis tabanlı yaklaşımı öğrenmek ve uygulamak amacıyla geliştirilmiştir.
